# PHPMyAdmin
> Important: durant l'installation, faire attention de bien cocher Apache2 en serveur


```sh
sudo apache2ctl stop
sudo apt-cache search phpmyadmin
sudo apt-get install mysql-server
apt-get install phpmyadmin
apache2ctl start
```
